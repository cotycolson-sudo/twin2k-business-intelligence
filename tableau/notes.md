# Tableau Analysis Notes

## Data Source
- Input: `data/processed/twin2k_flat.csv`
- Generated by: `scripts/extract_scores.py` from Twin-2K-500 persona parquet files

## Setup
1. Open Tableau Desktop and connect to `twin2k_flat.csv` (Text File connection)
2. Confirm field types: demographic dimensions as String, score measures as Number (Decimal)

## Suggested Dashboards
- **Demographics Overview**: Breakdown of participants by region, gender, age, education
- **Score Distributions**: Box plots or histograms for each psychological score variable
- **Cross-tabs**: Score means broken down by demographic group (e.g., Big Five by political affiliation)

## Notes
- `pid` should be set to Dimension > String (not aggregated)
- Use Measure Names / Measure Values for side-by-side score comparisons
- Filter out null score values when building visualizations to avoid distorting aggregates
