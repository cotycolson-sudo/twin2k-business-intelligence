# JMP Analysis Notes

## Data Source
- Input: `data/processed/twin2k_flat.csv`
- Generated by: `scripts/extract_scores.py` from Twin-2K-500 persona parquet files

## Setup
1. Open JMP and import `twin2k_flat.csv` (File > Open)
2. Confirm column types: demographic fields as Nominal, score fields as Continuous

## Analyses
- Use Analyze > Distribution for univariate summaries of each score variable
- Use Analyze > Fit Y by X for bivariate relationships
- Use Analyze > Multivariate for correlation matrices across score variables

## Notes
- `pid` is the participant identifier (nominal)
- Score variables with hyphens (`score_ST-TW`, `score_CNFU-S`) import correctly as column names
- Wave 1 vs Wave 2 conscientiousness: `wave1_score_conscientiousness` and `wave2_score_conscientiousness`
